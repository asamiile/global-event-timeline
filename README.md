## Global Data Timeline

A video visualization that maps extreme weather event CSV data onto a world map.

https://github.com/user-attachments/assets/a622e7a7-fe9a-43c3-99ac-6103c1aadb20

## Data Source

The extreme weather event CSV data is processed using the following Gist:
https://gist.github.com/asamiile/c0a57b6ac11f7a0707c67362222093af

Original data source:
>Japan Meteorological Agency
>https://www.data.jma.go.jp/cpd/monitor/extlist_s/

## Features

1. Open http://localhost:3000 to view the animation playback
2. Click "Export as Video" to download the visualization as an MP4 file

## Setup

- Installing dependencies (First time only)

```bash
pnpm i
```

- Starting the development server at http://localhost:3000

```bash
pnpm run dev
```
